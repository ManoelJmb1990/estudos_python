# Estudos em Python – 100 Days of Code

Este repositório documenta minha jornada de estudos no curso
**100 Days of Code: The Complete Python Pro Bootcamp**, ministrado por
**Dr. Angela Yu**.

O objetivo é praticar Python de forma consistente, desenvolvendo projetos
diários e consolidando fundamentos da linguagem.

## Objetivos
- Desenvolver lógica de programação
- Praticar Python diariamente
- Construir projetos reais
- Criar um histórico de evolução no GitHub

## Conteúdos estudados
- Python 3
- Automação e scripts
- Programação orientada a objetos
- Jogos e pequenos projetos
- Git e GitHub
